hexo 博客搭建中遇到的问题

##安装next主题
##1.1安装next

  • 安装next主题的方式很简单，利用git bash 在博客主目录下执行：
    git clone https://github.com/theme-next/hexo-theme-next themes/next
    然后打开站点配置文件，将theme属性默认值改成next
    theme: next(：后面要加空格)
    ##1.2选择自己喜欢的next主题样式
    scheme: Gemini
    ##2 更改站点属性
    将site字段，具体修改如下：
    title: 肉肉的博客 #网站标题(博客站点名字）
    subtitle: ‘起至今行’ #网站副标题，显示在标题下方
    description: ‘’ # 个人描述，
    keywords:
    author: 小二 #博客作者
    language: zh-CN #设置博客站点语言为中文
    timezone: ‘’ #设置时区，默认当前电脑时区
    ##3菜单栏
    ###3.1显示更多栏目
    在主题配置文件中修改如下：
    menu：
    home: / || home #首页
    about: /about/ || user #关于
    tags: /tags/ || tags #标签
    categories: /categories / || th # 分类
    ###3.2图标和内容量
    menu_settings:
    icons: true # 是否显示各个页面的图标
    badges: true # 是否显示分类/标签/归档页的内容量
